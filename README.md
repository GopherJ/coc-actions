# coc-actions

Actions menu for coc.nvim (neovim only)

![image](https://user-images.githubusercontent.com/5492542/71774253-78717700-2fa6-11ea-9629-1e88d0b5114e.png)

## Installation

`:CocInstall coc-actions`

**Commands**

- `:CocCommand actions.open`
